# ENGECON - Sistema de Gestão de Reformas Prediais

## Como Abrir o Projeto

### Opção 1: Usando o Live Server (Recomendado)

1. Abra o arquivo `index.html` com o **Live Server** do VS Code
2. O navegador abrirá automaticamente em `http://localhost:5500/`
3. Todos os links entre as páginas funcionarão perfeitamente

### Opção 2: Abrir Diretamente no Navegador

1. Procure o arquivo `index.html` na pasta principal do projeto
2. Clique com botão direito e escolha "Abrir com" seu navegador favorito
3. Todos os links funcionarão normalmente

## Navegação

### Menu Principal (Sidebar)
- **Resumo Geral** - Dashboard com panorama completo
- **Movimentações** - Acompanhamento de movimentações
- **Estoque** - Controle de estoque e financeiro
- **Obras** - Relatório e status de obras
- **Usuários** - Registro e gerenciamento de equipe
- **Compras** - Lista de colaboradores e compras
- **Relatórios** - Relatórios do sistema
- **Configurações** - Configurações de sincronização

### Links Adicionais
- Clique no logo **ENGECON** para voltar ao index.html de qualquer página
- No Portal de Login, existe um botão "Voltar" no canto superior esquerdo

## Recursos

- Design responsivo com Tailwind CSS
- Sistema de cores profissional (Midnight Blue + Signal Orange)
- Tipografia em Manrope para visual moderno
- Ícones do Material Symbols
- Navegação fluida entre todas as páginas

## Observações

Todos os links foram configurados para funcionar com caminhos relativos, garantindo que o projeto funcione independentemente de onde esteja armazenado.
